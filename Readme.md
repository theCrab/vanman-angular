## Hanami Angular
This is an Angular app from ground app in nodejs. The API is built with HanamiRB.

One of the many setups you could use.

### To run the app

```sh
$ git clone thisrepourl
$ cd vanman-angular
$ npm install
$ gulp
```

Visit `http://localhost:5000` and enjoy.
